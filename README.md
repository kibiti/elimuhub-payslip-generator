# Elimuhub Payslip Generator

A professional, reusable weekly payslip template for Elimuhub teachers, fully aligned with contract terms.

## Features

- **Interactive Form** - Fill in all details once and generate professional payslips
- **Automatic Calculations** - Gross earnings, total deductions, and net pay calculated instantly
- **Professional Design** - Clean, modern layout with Elimuhub branding
- **Contract Compliance** - Built-in compliance notes and terms
- **Print-Ready** - Optimized for printing/PDF export
- **Reusable** - Generate new payslips with one click
- **Auto-Save** - Form data is automatically saved in your browser

## How to Use

1. Fill the form with tutor details, hours, and rates
2. Click "Generate Payslip" to create the professional document
3. Print or save as PDF using the print button
4. Generate new payslip for next week with one click

The template automatically formats currency, dates, and calculates all totals. It's mobile-responsive and ready to use immediately!

## Sections Included

- Earnings breakdown with hourly rate calculation
- All deductions (transport recovery, security deposit, penalties)
- Payment details (status, method, date, transaction ref)
- Compliance warnings per contract terms
- Signature sections for both parties
- Contact information footer

## Contract Terms Implemented

- Transport advance recovery system
- Security deposit retention policy
- Friday payment cycle
- Client confirmation requirement
- Exclusive payment processing through Elimuhub
- Penalties for late arrival/unexcused absence
